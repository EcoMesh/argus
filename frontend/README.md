# Frontend

## Setup

### Requirements

+ [Node.js v18.x](https://nodejs.org/en/download/)
+ [Yarn](https://classic.yarnpkg.com/en/docs/install)

### Installation

```sh
yarn install
```

### Development

```sh
yarn start
```

### Build

```sh
yarn build
```
